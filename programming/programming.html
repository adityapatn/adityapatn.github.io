My Programming Journey

I started going to coding classes at the age of 11 (2021), where I learned block-based programming using Scratch. I also learnt programming in Python by age 12. I have a lot of experience programming in Python, although I still have a lot to learn. I love playing the video game Minecraft, and in 2023 I started dabbling with making Minecraft mods using the programming language Java. At this point I also started learning frontend web development using HTML, CSS, and JavaScript. During the 2023-2024 school year, I built a program for the VEX V5 Robotics competition in C++, although I am not as comfortable with is as I am with other languages.
